РИТМ: Аутстаф-процесс

Модуль подбора внешнего персонала с нуля. Объединил три роли пользователей в едином рабочем пространстве: руководителя-заказчика, рекрутёра-контрагента и лидера центра компетенций.

Роль Product Designer
Срок 7 месяцев
Команда PM, 2 Dev

О проекте

В рамках развития HR-платформы «РИТМ» разработал процесс подбора внешнего персонала (аутстаф) с нуля. Проект заменил стороннее коробочное решение и объединил три роли пользователей в едином рабочем пространстве: руководителя-заказчика, рекрутёра-контрагента и лидера центра компетенций.

Каждый этап процесса — от создания заявки до выхода кандидата на работу — проработан с учётом специфики взаимодействия ролей. Реализованы гибкие сценарии согласования, отбора кандидатов, назначения интервью и формирования кадрового резерва.

Задачи и вызовы

Ранее компания использовала стороннее коробочное решение для подбора аутстаф-персонала. Требовалось создать собственную платформу с учётом уникальных бизнес-процессов и интеграцией с внутренними системами.

Ключевые сложности:

  • 3 роли пользователей с разными правами и интерфейсами
  • Многоэтапное согласование заявок
  • Параллельные интервью с автораспределением по календарю
  • Встроенная коммуникация между участниками процесса
  • Автоматическое создание учётных записей

Решение

Спроектировал UX/UI для всех ролей, участвовал в подготовке ТЗ. Каждый этап процесса проработан с учётом специфики взаимодействия участников:

  • Проектирование форм создания заявок с валидацией данных
  • Карточки кандидатов с возможностью просмотра резюме и статусов
  • Система чатов для коммуникации между конкретными участниками
  • Назначение интервью с интеграцией календаря
  • Автоматическое создание профиля сотрудника при выходе на работу

Результаты

  • 113 экранов для 3 ролей пользователей
  • Замена стороннего решения, экономия нескольких миллионов рублей в год
  • Платформа выводится на рынок как B2B-продукт
  • Единственный Product Designer на проекте
